# Kinescope_Linker

## Активация
###Активация для: Google Chrome, Яндекс Браузер

1. Переходим по ссылке: chrome://extensions/
2. В правом верхнем углу есть ползунок "Режим разработчика". Если он отключен, то включаем его.
3. Появляется кнопка "Загрузить распакованное расширение". Нажимаем на нее и выбираем папку "Kinescope_Linker".
4. Готово! Теперь у вас появляются кнопки "Открыть трансляцию".
